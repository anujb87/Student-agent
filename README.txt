CAMPUS AGENT — PROTOTYPE
========================

HOW TO RUN:
1. Open Terminal
2. Navigate to this folder:
   cd /path/to/prototype
3. Start the server:
   node server.js
4. Open browser at:
   http://localhost:3001
5. Enter your Anthropic API key to launch

REQUIREMENTS:
- Node.js (any recent version)
- An Anthropic API key (get one at console.anthropic.com)

MODULES:
- Dashboard: Proactive nudges ranked by urgency
- Midterm Prep: Topic tracker, AI study guide, live quiz
- Course Selection: Professor comparison, degree progress, advisor chat
- Student Housing: Listings, filters, save/alert, group shortlist
