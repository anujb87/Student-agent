COURSE DESCRIPTION
================================================================
This course introduces the fundamental concepts of database management systems
with an emphasis on relational databases and SQL. Students will learn to design,
implement, and query databases for real-world business applications. Topics
include data modeling, relational theory, SQL programming, normalization,
transaction management, and an introduction to NoSQL systems.

================================================================
LEARNING OBJECTIVES
================================================================
By the end of this course, students will be able to:
  1. Design entity-relationship (ER) diagrams for business scenarios
  2. Translate ER diagrams into normalized relational schemas
  3. Write complex SQL queries including joins, subqueries, and aggregations
  4. Implement stored procedures, triggers, and views
  5. Understand indexing strategies and query optimization
  6. Describe basic NoSQL concepts and when to use them
