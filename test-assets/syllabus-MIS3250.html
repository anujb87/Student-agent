Course Description

This course introduces the fundamental concepts of database management systems with an emphasis on relational databases and SQL. Students will learn to design, implement, and query databases for real-world business applications. Topics include data modeling, relational theory, SQL programming, normalization, transaction management, and an introduction to NoSQL systems.
